# Email-Assistant-Project
Developed an AI-powered Email Assistant using Spring Boot and Gemini API that generates context-aware email replies. Built a Chrome extension to integrate directly with Gmail, enabling real-time response generation, and implemented RESTful APIs with secure environment-based API key handling.
